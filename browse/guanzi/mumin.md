# 牧民第一

:::details 牧民

1. 牧【养】民
2. 牧 放牧、管理（牲畜等），也引申为统治、治理（域等）。

个人理解：两者意思都有，偏后者

:::

## [国颂](#guosong)

&emsp;&emsp;凡有地牧民者，[务](#wu_01)在四时，[守](#shou_01)在仓廪[lǐn]。国多财，则远者来，地[辟举](#piju_01)，则民留处；仓廪实，则知礼节；衣食足，则知荣辱；上[服度](#fudu_01)，则六亲固；四维张，则君令行。故[省刑](#shengxing_01)之要，在禁[文巧](#wenqiao_01)，[守国](#shouguo_01)之度，[饰](#shi_01)在四维，[顺民](#shunmin)之[经](#jing)，在[明鬼神](#mingguishen)，[祇[zhī]山川](#zhishanchuan)，[敬宗庙](#jingzongmiao)，[恭祖旧](#gongzujiu)。不务天时，则财不生；不务地利，则仓廪不盈；[野芜旷](#yewukuang)，则民乃[菅](#jian)，上无[量](#liang)，则民乃[妄](#wang)。

```md [róng]
[róng]

2. 表示人的仪容、风度。
3. 接纳并容纳。
4. 宽容。参见“頌繫”。
5. 公开表达；明言。参见‘頌言’。
```

```md [个人理解]

```

<a id="mingguishen"></a>
:::: details 明鬼神 [míng guǐ shén]

:::code-group

```md [míng guǐ shén]
- 明：昭示、显明、尊崇

- 鬼神：天地神灵

要尊崇、彰明对鬼神的敬畏

（不是迷信，而是强调“敬天”）

百姓敬畏天地 → 不敢作乱
```

```md [孟凡君教授讲解]
管子不去回避鬼神，明鬼神
九十四年之后，孔子 “子不语，怪力乱神”，不说鬼神
子路去问鬼，孔子“未能事人，焉能事鬼”
```

```md [个人理解]
管子，孔子，对鬼神的态度
既是时代的变化，也是对像的区别
管子是对王说的，是对王和官员的要求，是从治理出发的
孔子，是对君子的要求，是从修身出发的
```

<a id="gongzujiu"></a>

:::details 恭祖旧 [gōng zǔ jiù]

- 祖旧：指父祖辈的旧交或故友。这里应该，既有长辈，也有旧制的意思

- 恭敬长辈遵守祖宗留下的制度

- 尊重祖先 → 不会轻易改变制度

- 守旧制 → 社会不混乱

:::

<a id="yewukuang"></a>
:::details 野芜旷[yě wú kuàng]

- 野：田野、土地

- 芜旷：荒芜空旷、无人耕种

- 民乃菅：百姓就像“菅草”一样（随便生长、无所依附）

如果田地荒废、不被治理，百姓就会像杂草一样散乱生长，没有依托，没有秩序。

:::

<a id="jian"></a>
:::: details 菅[jiān]

::: code-group

```md [jiān]
[jiān]

1. 一种多年生草本植物，叶细长且尖，根坚韧，可制炊帚、刷子。花绿色，果实为褐色颖果。
2. （Jiān）姓。

[guān]
春秋时期宋国的地名。
```

```md [孟凡君教授讲解]
这里菅是指懒惰的意思
```

```md [个人理解]
这里菅，野草应该和庄稼对比，
田野荒废，百姓就像野草，国家从百姓那里收获不到什么，
地辟举，百姓就像庄稼，国家可以从百姓那里有收获
```
