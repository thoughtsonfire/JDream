## 快捷键

功能|快捷键
:-:|:-:
加粗|Ctrl + B
斜体|Ctrl + I
引用|Ctrl + Q
插入链接|Ctrl + L

插入图片|Ctrl + G
提升标题|Ctrl + H
有序列表|Ctrl + O

## 基本语法

### 字体
- 斜体  
  *这里是文字*  
  `*这里是文字*`  
  _这里是文字_  
  `_这里是文字_`
- 加粗  
  **这里是文字**  
  `**这里是文字**`  
- 删除线  
  ~~这里是文字~~  
  `~~这里是文字~~`
- 粗斜  
  ***这里是文字***  
  `***这里是文字***`
