# 前言


## 简介


**什么是 JavaScript？**  


**JavaScript（JS）** 是一种广泛用于网页开发的编程语言，最初被设计为在浏览器中增强网页的交互性。如今它已发展为既能在浏览器中运行，也能用于服务器端开发（如 Node.js）的全栈开发语言。


**JavaScript 能做什么？**  


**在浏览器中：**

- 响应用户操作（点击、输入等）

- 动态修改网页内容（DOM 操作）

- 表单验证

- 控制动画、切换页面、弹窗等

- 与服务器通信（Ajax、fetch、WebSocket）

**在服务器中（通过 Node.js）：**

- 构建后端 API 服务

- 操作数据库

- 文件读写、脚本执行等
