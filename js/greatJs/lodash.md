# Loadsh  


## 描述  

`Lodash` 是一个 `JavaScript` 实用工具库，提供了一系列常用的函数来处理：

- 数组（`Array`）

- 对象（`Object`）

- 字符串（`String`）

- 函数（`Function`）

- 集合（`Collection`）

- 性能优化（如防抖/节流）

它的目标是：

>✅ 简化常见的 `JavaScript` 操作，避免重复造轮子，提高代码可读性与可维护性。

## 安装  

:::code-group
```sh [npm]
npm i lodash
```
```sh [pnpm]
pnpm i lodash
```
```sh [yarn]
yarn add lodash
```
:::
