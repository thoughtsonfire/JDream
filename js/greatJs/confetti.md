# 纸屑  


## 描述  

纸屑、雪花类特效，用起来非常方便

## 安装  

:::code-group
```sh [npm]
npm i canvas-confetti
```
```sh [pnpm]
pnpm i canvas-confetti
```
```sh [yarn]
yarn add canvas-confetti
```
:::
